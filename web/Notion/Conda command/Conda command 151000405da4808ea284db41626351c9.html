Conda command

使用條件：已安裝完 conda

  • 版本 & 更新
conda -V
conda update conda # conda upgrade

  • 建立虛擬環境 (name={myenv})，並安裝package 
    conda create -n {myenv} {package}

  • 列出所有虛擬環境
conda env list # conda info -e
  • 啟動、退出環境
conda activate {myenv}
conda deactivate
  • 搜尋、安裝、更新、刪除套件
    conda search {package}
conda install {package}
    # -c: channel, 如果找不到，可以到 conda-forge 搜尋
    # conda install -c conda-forge {package}
conda update {package}
conda remove {package} # conda remove -n {myenv} {package}
  • 顯示安裝的package
conda list {package}
conda list -n {myenv}
  • 刪除虛擬環境
    conda env remove -n {myenv} # conda remove -n {myenv} --all
  • 匯出檔案建立環境
    • txt 檔
    conda list -e > {myname.txt}
conda create -n {myenv} -f {myname.txt}
    • yml 檔
    conda env export -n {myenv} -f {myname.yml}
conda env create -f {myname.yml}

安裝 tensorflow-gpu版

版本參照（語言選English，較多選擇）：https://www.tensorflow.org/install/source

# 先找{version}，再安裝
# 基本 cudatoolkit, cudnn, compiler 會一起自動安裝，需要特定版本則需先安裝
conda search -c conda-forge tensorflow-gpu
conda install -c conda-forge tensorflow-gpu={version}
